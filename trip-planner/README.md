# Trip Planner (React + Vite + TypeScript)

This is a scaffolded starter for the Trip Planner project: a static React + Vite frontend deployed to GitHub Pages, Firestore for data, and Firebase Cloud Functions as LLM proxy.

What's included:
- React + Vite + TypeScript app
- Tailwind CSS
- Firebase folder with example Cloud Function (LLM proxy)
- GitHub Actions workflow to build and deploy to GitHub Pages

Next steps:
1. Review and customize firebase project settings and secrets.
2. `npm install` then `npm run dev` to run locally.
3. Configure GitHub repository and push; GitHub Actions will deploy to Pages on push to main.
